Tamagawagakuen (たまがわがくえん) is a name of a street of Machida-shi, Tokyo. The current administrative place name 1, Tamagawagakuen - 8 (house indication area). The zip code is 194-0041. The population 16,499 people (as of Basic Resident Register September 1, 2016 [2]). There is educational foundation Tamagawagakuen in the town.
