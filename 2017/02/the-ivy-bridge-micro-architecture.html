The Ivy Bridge micro architecture

The Ivy Bridge micro architecture

The Ivy Bridge micro architecture (the ivy bridge micro architecture) is the microarchitecture of the microprocessor developed by Intel.

シュリンク did the Sandy Bridge micro architecture that was a forerunner to 22nm and was developed, and manufacture was announced as the third generation Core processor formally on April 24, 2012.

Function improvement

Ivy Bridge in comparison with Sandy Bridge including the improvement of the following functions [1] [2].

  • 3D try gate transistor (there is little power consumption 50% at the same performance level by the comparison with the 2D plane transistor) [3]
  • PCI Express 3.0 [4].

A note: The CPU less than Core i5 does not support PCI Express 3.0; in PCI Express 2.0 is worked

  • The drive clock magnification of the CPU up to 63 (in Sandy Bridge 57) [5]
  • The DRAM improves 200MHz in a maximum of 2,800 MT/s [5]
  • Intel HD Graphics 2500/4000。 DirectX 11、OpenGL 3.1、OpenCL 1.1を support[6]
  • Embedded GPU has a maximum of 16 practice units (EU). Sandy Bridge a maximum of 12 [7]
  • New random number generator and RdRand order [8]. Code name Bull Mountain[9]
  • Reinforcement of REP MOVSB and the STOSB operation
  • Addition of the half precision floating point conversion order
  • Reinforcement of the hardware pre-fetch
  • Improvement of micro-op queue in the singles red
  • I can operate the movement of the register in the 0 lei ten sea
  • Speedup of floating point division and the square root operation
  • Speedup of ROL and the ROR order
  • DDR3L and setable TDP[10] for mobile processors
  • Plural 4K animation reproduction
  • Intel Quick Sync Video[7]
  • 3 screen output support [11] of embedded GPU

Intel 7 series chip set adoption motherboard is necessary to validate all functions. It was announced to the Intel 6 series chip set adoption motherboard of the thing which function restrictions produced that the deployment was possible [12].
