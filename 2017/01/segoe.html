Summary

Microsoft uses Segoe in online, both print commerce mediums; in that late years [when?] The logos of the product of the released company are included. In addition, Microsoft uses Segoe for a company name logo of the company [2].

In addition, the derivation font group of Segoe called Segoe UI (font family) is used in a product of various Microsoft, and this font is used for application made in the company such as Microsoft Office 2007 and Windows Live Messenger 2009. In addition, I am set as System Font of the defaults by Windows Vista and Windows 7 of the English version. As of 2013, Segoe UI is used in the user interface of the Outlook .com introduced as Web email software for Hotmail. Segoe and Segoe UI are fonts for Roman letters, Greek characters, Cyrillic and the Arabic alphabet, and グリフ such as Japanese or Chinese is still less included. The System Font of the default of Japanese edition Windows 7 becomes May Rio [1].

Originally Segoe was developed by Monotype (English version) company, but, as for the name, Microsoft holds a trademark now.

History

Segoe was devised in those days by Steve Matteson which worked for Monotype company (as for the name Agfa Monotype). Microsoft entered into licensing agreement to use as a font of a typeface and the user interface and it warm and was designed as the letter which it was easy to read. Matteson produced various style of handwriting and italic types from Segoe.

License debate

For 2,004 years, Microsoft performed the registration application of a registered trademark and the design patent in European Union (EU) saying that it was an original font in a specific font of Segoe and the italic type font. Linotype company which was a German font production company protested it to be able to do it when Segoe UI closely resembled it with a font called Frutiger which Linotype company developed. I insist that Microsoft was not able to show that Linotype released Frutiger and the derivation font before 2004 in patent application to EU.

EU rejected these claims and rejected patent application of Microsoft in February, 2006 [3]. Microsoft did not appeal for this judgment. Microsoft applies a design patent of the United States of America to various Segoe-based fonts now.

Around the end of 2004 when it is the same time, Monotype which was a subsidiary of Agfa is purchased in TA Associates for six years and will run a company name some other time with Monotype Imaging. Monotype Imaging purchases Linotype afterwards in August, 2006 [4]. Therefore, Linotype company which made an objection against the legitimacy of the Segoe patent of Microsoft at the end of 2006 was a wholly owned subsidiary of Monotype Corporation which consented to Segoe at first in Microsoft.

 
It is difference in Segoe UI, Frutiger, Segoe (from the top sequentially)

I adopted totally different form by some letters in Segoe UI and Frutiger, and a big difference came to be reflected in use in the screen of the low resolution than Frutiger. However, pointed out Ulrich Stiehl[5] that most of these differences were introduced in a new version of Segoe UI, and early stage of Segoe UI closely resembled Frutiger [6]; [7].

Simon Daniels which summarized a typography group of Microsoft in November, 2005 stated, "it was not a thing made by hand of Microsoft, and the early stage of design depended on Monotype Corporation which did licensing agreement for Microsoft and customized the original Segoe font as needed in the deployment process to different process, application and device and expanded the expansion of the font" [8].

In April, 2006, the spokesman of Microsoft spoke it as follows.

"I hold the latest license which I used Segoe font that Segoe was developed as an original font in 2000 by Agfa Monotype (current Monotype Imaging) company, and we acquire original Segoe font in 2003, and to develop the expansion font of the group with the name of Segoe, and most of these new fonts are protected in the United States of America by a design patent, and Segoe is not a font derived from Frutiger, and Microsoft can distribute certain Frutiger font when I use Microsoft product including Office and Windows, and there is, and there is clear difference in Segoe and Frutiger. In addition, unlike the style of handwriting that performed a clone, the font group of Segoe does not have the compatibility that I can use as a substitute of Frutiger. 」[9]

Under the Copyright Act of the United States of America, a copyright is not applied to the form of the abstract letter of the function text font, and Copyright Act is applied only to the programming cord included in the font. This enabled making and the distribution of the clonal font.

Segoe UI was one of the Segoe font group used for a text for user interface (UI) in a product of Microsoft and improved it so that it was possible as possible to read a sentence written by any language like a function to perform Assistant online user. It is distinguished from Lucida Grande which is previous Tahoma (English version) and the UI font in the Mac OS. Monotype Imaging developed Segoe UI [10].

"Segoe UI Light" and "Segoe UI Semibold" were introduced by Windows 7 for the first time [11].

 
Clear difference in old version new version (bottom) of Segoe UI

In Windows 8, some changes and stylistic changes are made to Segoe UI [12]; [13] [14] [15].

  • Semilight version was introduced to assume a stock size of light 11 pixels.
  • The variant character of the truth italic type was introduced for Light, Semilight, Semibold version.
  • Adjustment was made for improvement of the quality when I read a letter on the screen in Segoe UI Light and Segoe UI Semibold.
  • The change of the typography that was extremely similar to WP font group was carried out. A remarkable difference is seen in 1, 2, 4, 5, 7, 8 by Windows Vista and Windows 7 in particular "I" and "Q".
  • A Hebrew letter, an Armenian letter, a Georgian letter, a letter such as the initial Cyrillic (English version) came to be supported newly.
  • Variant form of the OpenType is included.

Letter

Segoe UI is optimized for the ClearType rendering environment that is default setting of Windows Vista. Therefore, except size (8, 9, ten points) of UI where Segoe UI is performed font hinting of by the rendering of the double level, a readability may be spoiled when ClearType becomes invalid. A standard font size increases to 9-point by Windows Vista to make it easy to improve a layout by all languages, and to read.

Roman letters, Greek characters, Cyrillic, Unicode 4.1 coverage including the Arabic alphabet are introduced into the Windows Vista version (version 5.00) of Segoe UI, and 2843 グリフ in total is included by the thickness of the normal letter.

Segoe UI implements an italic type unlike a diagonal character used in Frutiger and Helvetica.

Kind

  • Segoe UI Mono is a monospaced font in Segoe UI. This supports the signs such as Roman letters (including Eastern Europe and the Turkish letter), Greek characters, Cyrillic, a Hebrew letter, a Thai letter, the map sign. This font has a font of two kinds of density, but the italic type is not implemented.
  • Segoe UI Symbol is a font including a chessman and soot, the dice, the mathematical character of the card, an arrow sign, the sign such as OCR, Braille, an ogham, the rune [16]. However, this is a font encoded in Unicode using the sign assigned for code point of this Unicode in substitution for the font that encoded a sign [17]. The OCR letters such as the sets of the writing style that is available for access by Segoe UI and the OpenType of the monospace are included in this font. In addition, with the bug that グリフ of U+2546 becomes same as U+2545 (from Windows7 all the versions of the Windows10 preview release).

Use

The Segoe UI font group is used as a font of Microsoft Office 2007 and Microsoft Office 2010, Windows Vista and Windows 7. The Segoe font that is not Segoe UI is included in Windows XP Media Center Edition 2005 in form for the Trebuchet MS.

The "Segoe UI" font installer of the simple substance for Windows XP is not opened to the public, but comes to be installed automatically when I install Windows Live Messenger and Windows Live email. In addition, when I install Microsoft Word Viewer and Microsoft PowerPoint Viewer 2007, 1 version of the Segoe font is installed automatically, but this is also a font unlike Segoe UI.

The "Segoe UI Semilight" font installer of the simple substance for Windows 7 is not offered, but comes to be installed automatically when I install Microsoft Office 2013.

Early version of Segoe, probably evaluation copy is included in a specific version of SuSE Linux, but cease to function for of the OS. The first company using Segoe was Canadian illumination signboard company Scala under the license of Monotype company (the association does with FF Scala of the font).
